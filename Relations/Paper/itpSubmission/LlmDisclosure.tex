\section{LLM Disclosure}

Large language models (LLMs) proved of limited utility for authoring Agda code, often producing syntactic and logical errors. They have not been used to make any substantial contribution to the code base.

% In the latter stages of the project, however, their performance improved sufficiently that one author employed GPT‑5.3 Codex to assist with syntax corrections when developing helper lemmas in \texttt{TRS-examples.agda}. The LLM was used solely as an aid for drafting and error correction; it did not determine the design, proofs, or final content in the code base.
